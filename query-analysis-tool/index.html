How to use

Format your .csv file so that the data is in the following order:
Search terms | Impressions | Clicks | Cost | Conversions | Conversion Value
Delete any extra rows above your headers/data (Headers are optional)
Once your .csv file is formatted, go ahead and upload it then click submit to run.
