Alphanumeric Query Analysis Tool

This web app was originally built for analyzing Google Ads query data. Typically queries containing alphanumeric terms like "XRM2400" see a higher purchase likelihood (conversion rate or CVR) and return on ad spend (ROAS) typically because they refer to specific model numbers. This tool allows the user to analyze search query data and to export a list of all these alphanumeric terms for the sake of targeting them in ad campaigns.

How to use

Format your .csv file so that the data is in the following order:
Search terms | Impressions | Clicks | Cost | Conversions | Conversion Value
Delete any extra rows above your headers/data (Headers are optional)
Once your .csv file is formatted, go ahead and upload it then click analyze to run.
